Project 2: Python Hangman Game

This Python project is a text-based version of the classic game "Hangman." It welcomes the player and provides them with ten chances to guess a hidden word. The player inputs guesses, with incorrect ones deducting from their turns. The game ends with either a victory message for a successful guess or defeat when turns run out. It's a fun way to engage with fundamental programming concepts while enjoying the challenge of word guessing.
